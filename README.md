# PRO-ΠΑΙΔΕΙΑ

Ένα διαδραστικό παιχνίδι εξάσκησης στην προπαίδεια για παιδιά, με γρήγορο ρυθμό, ήχους, animations και σύστημα high score.

## Live Page

[Παίξε το παιχνίδι εδώ](https://dmcode9.github.io/MultiplicationsGame)

## Τι Είναι

Το `PRO-ΠΑΙΔΕΙΑ` είναι ένα browser game για εξάσκηση στον πολλαπλασιασμό.
Ο παίκτης προσπαθεί να απαντήσει όσο το δυνατόν περισσότερες σωστές πράξεις μέσα στον διαθέσιμο χρόνο.

## Πώς Παίζεται

1. Διαλέγεις επίπεδο.
2. Διαλέγεις τρόπο παιχνιδιού:
   `Επιλογές` ή `Γράψιμο`.
3. Πατάς `ΠΑΙΞΕ!`.
4. Απαντάς στις πράξεις όσο πιο γρήγορα και σωστά μπορείς.
5. Στο τέλος βλέπεις το σκορ σου και αν έκανες νέο ρεκόρ.

## Τρόποι Παιχνιδιού

### Επιλογές

Ο παίκτης βλέπει την πράξη και διαλέγει τη σωστή απάντηση από τις διαθέσιμες επιλογές.

### Γράψιμο

Ο παίκτης πληκτρολογεί μόνος του τη σωστή απάντηση με το αριθμητικό πληκτρολόγιο της οθόνης.

## Χαρακτηριστικά

- Επίπεδα από `1` έως `12`
- Ειδικό επίπεδο `Όλα`
- Ξεχωριστά high scores ανά επίπεδο και ανά τρόπο παιχνιδιού
- Ήχοι και οπτικά εφέ
- Animations για σωστές και λάθος απαντήσεις
- Bonus `+50` πόντων κάθε 5 συνεχόμενες σωστές απαντήσεις
- Celebration animation για νέο ρεκόρ
- Responsive σχεδίαση για κινητό και υπολογιστή

## Σύστημα Βαθμολογίας

- Κάθε σωστή απάντηση δίνει `+10` πόντους
- Κάθε λάθος απάντηση αφαιρεί `-5` πόντους
- Κάθε 5 συνεχόμενες σωστές απαντήσεις δίνουν επιπλέον `+50` bonus

## Σε Ποιους Απευθύνεται

Το παιχνίδι είναι σχεδιασμένο κυρίως για παιδιά σχολικής ηλικίας που θέλουν να εξασκηθούν στην προπαίδεια με πιο ευχάριστο και παιχνιδιάρικο τρόπο.

## Τεχνολογίες

- `HTML`
- `CSS`
- `JavaScript`
- `localStorage` για αποθήκευση ρεκόρ και επιλογών
- `canvas-confetti` για τα confetti effects
